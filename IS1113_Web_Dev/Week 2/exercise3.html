Formatting content

In XHTML, there are a number of different elements that change the apperance of the page's content when used. Some tags will render the content in exactly the same way but their meaning is different. After all, XHTML is primarly a way of structuring the content of a document. In later chapters we will see better ways of formatting content. For now, though, the tags that we learned in this Lab will suffice.

Special Characters and Formatting:

Some characters in XHTML have a special code to render them on the browser. If you want to render tags in the browser we use the "less than" and "greater than" symbols e.g. <strong></strong>. There are other characters such as the ampersand &, the copyright © and the euro: €
Then there are tags for adding subscript and for adding superscript and tags for putting a strikethrough the content.
